# casetech-assist

AI troubleshooting assistant built with Next.js.

## Development

Run all commands from the repository root:

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Environment

Local environment files are ignored by git. Keep `.env.local` in the repository root.

Google service account key files are also ignored by git. If you use
`GOOGLE_APPLICATION_CREDENTIALS`, point it at a local key file from the root of
this repository.
